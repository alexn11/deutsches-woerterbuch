Translation dictionaries, extracted from Wiktionary

The data itself is released under the CC-BY-SA license and can be freely used under the same license.

How does it work?

Simply download the page to your device and open it in your browser. You can then search for any word in the dictionary. Except for accessing the linked entries, no internet connection is required to use the dictionary, as it is stored locally on your device. No need for an app or any special software. Just open the HTML file in your browser and start searching for words.

Available languages

Since the data come from Wiktionary, this limits the available languages to those that are present in the Wiktionary database. I have extracted the translations for a few languages of my choosing. If you want to create a dictionary for another language, you can do so by downloading the data from Wiktionary and running the scripts attached to the project (see the documentation of the project for more information). I've put the languages I have extracted in three categories:
  • Languages with a sufficient number of entries ("usable"): these have enough translations (tens of thousands) to be used like a real dictionary,
  • Languages with an intermediate amount of entries ("interesting"): the number of translations is insufficient for a general use, but it might be okayish for basic vocabulary,
  • Languages with not so many entries ("experimental"): language with less than a thousand (or so) translation. I've included them for fun.
